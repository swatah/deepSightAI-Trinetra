## What is AGPL v3?

AGPL v3 is a **copyleft free software license** that ensures:

- **Freedom to use** for any purpose
- **Freedom to modify** and adapt the software
- **Freedom to share** modifications with others
- **Freedom to distribute** copies

The "Affero" clause specifically ensures that if you run the software on a server and let users interact with it over a network (SaaS model), you must also make the modified source code available to those users.

## Why AGPL v3?

We chose AGPL v3 to:

1. **Ensure openness** - Everyone benefits from improvements
2. **Prevent proprietary forks** - No one can take deepSightAI Trinetra closed-source
3. **Protect users** - Users retain their freedoms even in SaaS context
4. **Encourage contributions** - Community improvements flow back to all

This license is **business-friendly** for:
- On-premise deployments (no obligation to share internal mods unless distributed)
- Service providers offering managed deepSightAI Trinetra hosting (must share modifications)
- Consulting firms customizing for clients (modifications go back to client)
- Academic research (open science)
